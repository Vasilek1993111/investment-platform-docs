# investment-platform-docs
<h2>Описание:</h2>  
Система InvestmentTradingService представляет собой автоматизированный помощник для торговли на российском фондовом рынке, интегрированный с Tinkoff Invest API.

Платформа мониторит рыночные данные в реальном времени, сканирует инструменты (акции, облигации, фьючерсы MOEX) и выявляет торговые возможности по заданным стратегиям — сканеры ищут паттерны вроде пробоев, лимитов или аномалий объема.

<h2>Ключевые функции:</h2> 

<b>Сбор данных:</b> Scheduler загружает котировки, стаканы и инструменты через Tinkoff API в БД PostgreSQL.


<b>Анализ:</b> Автоматические сканеры (market/instruments) применяют правила для поиска сигналов.

<b>Уведомления:</b> Alerts отправляют детальные нотификации в Telegram-канал с графиками и рекомендациями.

<b>Автоматическая торговля:</b> Выставление отложенных заявок (limit orders) по срабатыванию сигналов через Tinkoff OrdersService.

<b>Полуавтоматическая торговля:</b> Подтверждение рыночных заявок (market orders) пользователем через API после уведомления.

<b>Ручное исполнение:</b> Полный контроль над немедленными сделками остается за пользователем.
